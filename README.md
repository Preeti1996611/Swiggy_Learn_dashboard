# Swiggy_Learn_dashboard
Project title / headline:

Swiggy Analytics: Restaurant & Delivery Insights Dashboard

Short description / purpose:

This interactive Power BI dashboard uses a Swiggy dataset from Kaggle to analyze restaurants and delivery listings across Indian cities. It helps users explore cuisines, ratings, prices, and delivery performance to support data‑driven decisions for marketing and operations teams.

Tech stack:

Power BI Desktop – dashboard creation and data visualization.
Power Query – importing and cleaning Kaggle CSV files.
DAX – measures such as total restaurants, average rating, cost metrics, and filtered KPIs.
Power BI data modeling – relationships between tables (restaurants, locations, categories, etc., if your dataset has them).

Features / highlights of the dashboard:

City overview page showing total restaurants, average rating, and distribution of cuisines for each city.
Cuisine analysis page comparing popular cuisines by count of restaurants, average rating, and typical “cost for two”.
Price & rating insights showing how restaurant ratings vary by price range (budget, mid‑range, premium).
Filters/slicers for city, cuisine, veg/non‑veg, and rating band to allow interactive exploration.
Map visual displaying restaurant locations using latitude and longitude if available in your dataset.
Top‑N views such as “Top 10 highest‑rated restaurants per city” or “Top 10 most expensive restaurants”.

Source data – Swiggy restaurant/cleaned dataset downloaded from Kaggle.
